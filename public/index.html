API Documentation - Hotel References

API untuk mengakses data referensi hotel di Kota Padang

Base URL: https://padang-hotel-api.vercel.app/

GET /hotels

Mendapatkan daftar hotel di Kota Padang

Query Parameters

Parameter Tipe Deskripsi
search optional string Mencari hotel berdasarkan nama
star_rating optional number Filter berdasarkan bintang (0-4)
limit optional number Jumlah data per halaman (default: 10)
page optional number Nomor halaman (default: 1)

Contoh Penggunaan

Mencari hotel bintang 4:

GET /hotels?star_rating=4

Mencari hotel berdasarkan nama:

GET /hotels?search=mercure

Status Codes

Status Deskripsi
200 Success
400 Bad Request - Parameter tidak valid
404 Not Found - Data tidak ditemukan
500 Internal Server Error
